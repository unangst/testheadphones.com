Tech Theory


With the increasing number of web sites transitioning to serve content over secure (https:) connections
(now including YouTube.com, Google.com, Pandora.com, etc.), many organizations need a benign web page to indicate
when a filter has been bypassed successfully.

We recommend that you block this site and direct your users to visit FilterOverride.com
to bypass your organization's web filter via a block page of your choosing. After doing so, they'll see a "Congrats!" type message,
indicating a successful authentication / bypass / override of your web filter.

FilterOverride.com is permanently hosted via http:// on Github and should have limited downtime.
