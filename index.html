Tech Implementation


We recommend that you block FilterOverride.com and and direct your users to visit FilterOverride.com
to bypass your filter's block page. Once they have done so, they see sample "Congrats!" type message
which indicates a successful authentication / bypass / override of your web filter.

With the increasing number of web sites transitioning to serve content over secure (https:) connections
(now including YouTube.com, Google.com and Pandora.com), many organizations need a benign web page to indicate
that the filter bypass has been successfully.

FilterOverride.com is permanently hosted via http:// on Github (read: no bandwidth concerns).
